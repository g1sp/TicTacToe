# Tic Tac Toe Game

A simple and interactive Tic Tac Toe (X and O) game built with HTML, CSS, and JavaScript.

## Features

- **Interactive Game Board** - Click cells to place your mark
- **Turn Indicator** - Shows whose turn it is (X or O)
- **Win Detection** - Automatically detects when a player wins
- **Draw Detection** - Identifies when the game ends in a draw
- **Smooth Animations** - Hover effects and transitions for better UX
- **Responsive Design** - Works seamlessly on mobile and desktop devices
- **New Game Button** - Restart a new game anytime

## How to Play

1. Player X always goes first
2. Players alternate by clicking on empty cells
3. The first player to get three marks in a row (horizontal, vertical, or diagonal) wins
4. If all cells are filled with no winner, it's a draw
5. Click "New Game" to play again

## Winning Combinations

- **Rows**: Top, Middle, Bottom
- **Columns**: Left, Middle, Right
- **Diagonals**: Top-left to Bottom-right, Top-right to Bottom-left

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Gradients)
- Vanilla JavaScript
